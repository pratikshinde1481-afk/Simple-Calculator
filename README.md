# Simple Calculator in Java

This project is a **console-based Simple Calculator** developed using **Java**. It performs basic arithmetic operations like addition, subtraction, multiplication, and division.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Menu-driven program
* User input using Scanner

## Technologies Used

* Java
* JDK
* Command Line Interface

## How to Run

1. Clone the repository
2. Open the project in any Java IDE (Eclipse, IntelliJ, VS Code)
3. Compile and run the `SimpleCalculator.java` file
4. Follow the menu instructions to perform calculations

## Example

Select an operation from the menu and enter two numbers to get the result.
